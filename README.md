# ElKoubasy Group - Corporate Website

A premium, high-performance corporate website for **ElKoubasy Group for Food Industries**. Built with a focus on cinematic storytelling, mobile responsiveness, and high-quality visuals.

## 🚀 Experience
- **Cinematic Hero**: Transparent high-end product visual with parallax effects.
- **Dynamic Animations**: Powered by GSAP for smooth, premium transitions.
- **Multilingual Support**: Fully integrated English and Arabic versions.
- **Optimized for Mobile**: A "very professional" smartphone experience with tailored layouts.
- **Modern Stack**: Next.js (App Router), Tailwind CSS v4, and TypeScript.

## 🛠️ Technology Stack
- **Framework**: Next.js 15+
- **Styling**: Tailwind CSS v4 (vanilla CSS + @theme)
- **Animations**: GSAP (GreenSock Animation Platform)
- **State Management**: React Context (Language, Theme)
- **Icons**: Lucide React & Custom SVG

## 📁 Getting Started

### Prerequisites
- Node.js 20+
- npm or pnpm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Project Structure
- `/app`: Main application routes and page structures.
- `/components`: Reusable UI elements (Navbar, Footer, SectionHeading, etc.)
- `/lib`: Helper functions and animation configurations.
- `/public`: High-resolution assets and brand imagery.

## 🚢 Deployment
The project is optimized for deployment on **Vercel**:
1. Connect your GitHub repository to Vercel.
2. The default build settings will automatically handle the Next.js optimization.
